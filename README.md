# Node Firebase

## Getting Started

Clonar el repositorio

```
git clone https://github.com/AlexanderPrincipe/Node-Firebase.git
```

Entrar a la carpeta del proyecto

```
cd Node-Firebase
```

Instalar todas las dependencias

```
npm install
```

Correr el proyecto

```
npm run dev
```
